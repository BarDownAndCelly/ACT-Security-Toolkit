# AI Security Prompt Playbook

A collection of prompt engineering techniques applied to real-world cybersecurity workflows.

This repository demonstrates how modern AI prompting techniques can be used to improve security operations, incident response, insider threat investigations, threat intelligence analysis, data loss prevention (DLP), and Microsoft Purview workflows.

## Objectives

* Demonstrate practical prompt engineering skills
* Apply AI to real cybersecurity use cases
* Showcase structured prompting methodologies
* Build a reusable library of security-focused prompts

## Prompt Engineering Techniques

* Zero-Shot Prompting
* One-Shot Prompting
* Few-Shot Prompting
* Chain-of-Thought Prompting
* Prompt Templates

## Security Use Cases

* Insider Threat Investigations
* Incident Response
* Threat Intelligence
* Data Loss Prevention (DLP)
* Microsoft Purview
* Security Operations Center (SOC) Analysis
